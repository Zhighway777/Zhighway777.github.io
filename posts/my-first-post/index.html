第一篇博客用于记录博客的搭建过程 ##模板 博客使用hugo工具进行搭建与管理，hugo提供了丰富的tamplates和draft功能使得博客网站的开发更加简单，开发者这需要关注博客的内容本身，而不必特地学习CSS或HTML进行网站搭建，除此之外，博客内容使用Maekdown格式，更方便不同平台的移植。 ##hugo 使用方法 具体参阅Hugo-quick-start

博客内容在/content/posts文件夹下，以*.md的格式存放，每增加一个*.md文件，则在博客中增加一个子页面。 在*.md开头部分有博客的基本信息：题目，日期，和draft状态。 drift默认被设置为ture,说明此时是草稿状态。 保存draft后，在命令行使用 hugo server -D可以打开draft,进行实时修改 当修改后的内容符合要求，可以将draft调整为false，即可保存修改并对修改后的内容进行publish 最后使用hugo命令发布网站
