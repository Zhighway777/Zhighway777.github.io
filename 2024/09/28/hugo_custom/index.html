在Hugo的Markdown中直接插入HTML

方法：使用hugo shortcode实现

hugo处理短代码时，首先搜索layouts/shortcodes中的相关文件，在到theme/layouts/shortcodes中搜索相关文件。

为了自定义shortcode, 首先在layouts/shortcodes中新建一个insert_html.html文件，内容只需要一行：

{{ .Inner}}

使用shortcode

上述创建的文件名称就是 shortcode 的名字，因此已经创建了一个名为 insert_html 的 shortcode 我们在 Markdown 里使用这个 shortcode 插入一段 HTML 代码， 将HTML的内容放在 \{\{<insert_html>}} 和 \{\{</insert_html>}} 之间。

\{\{<insert_html>}}
	<html 全部内容>

\{\{</insert_html>}}

注意： ‘{’ 前的 ‘\’使用来反转义的，在使用的时候需要去掉 ‘\’
