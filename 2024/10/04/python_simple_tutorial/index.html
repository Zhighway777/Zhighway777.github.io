Common Functions

print()

Print the characters into the screen.

input()

等待用户在键盘上输入一些文本，并按下回车键，将此文本作为字符串赋值给一个变量。

len()

得到len中字符串字符的个数

str(), int(), float()

str():将输入的类型转为字符串类型 int():将输入的类型转为整型 float():将输入的类型转为浮点型

round()

round( x, n) 返回{浮点数} x 的四舍五入的数值， n为考虑的小数点的位数

and, or

“二元”操作符， 接受两个表达式或者布尔值

not

“一元”操作符， 接受一个布尔值或表达式。求相反的布尔值

Control flow

code block

代码块的三个规则：

  1. 缩进为代码块的标识符
  2. 代码块可以嵌套
  3. 缩进减少为0, 或者减少为外围代码块的缩进，代码块结束

if sentence

语法格式

if condition1 :
	content 1
elif condition2 :
	content 2
else :
	content 3

while

语法格式: 满足 condition,子句执行，不满足 conditon, 子句跳过。

while condition:
	sub codes

for and range()

for的语法格式：

for i in range(number):
	content
#等价于C中 for(i=0; i < 5; i++)

range()

这里第一个是变量起始的数值，第二个变量是上限（不包括），第三个变量是步长。步长可以是负数，这时起始数值(start)应该大于上限(stop)。

range(start, stop, step)

sys.exit()

使得程序提前终止或者退出（同时输出此时的content信息，方便debug）

import sys

while True:
	print('Type exit to terminate the program')
	response = input()
	if response == 'exit':
		sys.exit()
	print('You typed' + response + '.')
