Common Functions

在这里简单的介绍一下python的函数是有必要的。 在python中，存在很多内置函数，可以直接使用，如print()函数。用户也可以自行定义一些函数，来实现一个具有一定功能的代码块。函数可以有参数，也可以沒有参数。对于有参数的函数，我们需要对参数进行传递。

##参数传递 python的函数和C语言作为对比，C语言中，参数需要指定类型，而且有型参和实参之分。但是在python中，只有对象(object)采用有“类型”这一特征，变量是没有类型的。

a = [1, 2, 3]

a = "Zeng"

在这里，需要把a理解为一个对象的引用(对象的指针)，可以指向List对象， 也可以指向String类的对象。

可变对象(mutable)和不可变对象(immutable)

在python中， 三个不可更改的对象：

strings
tuples
numbers

两个可更改对象：

List
Dict

那么对于python的参数传递而言： mutable 可变类型：类似于C的引用传递，实参。func内部对参数的修改也会改变外部的该参数所对应对象的数值。例如

def change_int(a):
	a = 10
# new variable b:
b = 2
print (change_int(b))
# Output 2

immutable 不可变类型：类似于C的值传递，传递的只是该参数的数值，不会修改参数所对应的对象本身。

def change_list(my_list):
	my_list.append([1, 2, 3, 4])
	print "Value in function: ", my_list
	return

#Call the function:
my_list = [10, 20, 30, 40]
change_int(my_list)
print "Value out of function: ", my_list	
#Value in function: [10, 20, 30, 40, [1, 2, 3, 4]]
#Value out of function: [10, 20, 30, 40, [1, 2, 3, 4]]

可见函数确实会改变外部mutable的值。 一下是对可变对象和不可变对象的进一步解释：

  1. 不可变对象的引用是可变的，在对不可变对象进行修改时，实际上是创建了新的对象，然后将“引用”指向了这个新的对象。
i = 73
i += 2

2. 可变对象的内容可以变化，但是变量的对象引用不会发生改变。

m = [5, 9]
m+=[6]

参数的类型

必备参数， 关键字参数， 默认参数， 不定长参数

  1. 必备参数就是在调用函数是，使用函数定义时的参数顺序来进行传递，需要保证参数顺序的一致性，否则容易出错。

  2. 关键字参数，通过在调用时使用关键字来表明传递的是哪一个参数，这样有助于清晰的进行参数传递。

  3. 默认参数， 如果参数的数值没有传入，则使用默认参数作为该参数的值。

  4. 不定长参数， 使用在一个函数可以处理的参数数量比声明时更多的情况下。注意，在声明的时候不会进行命名。

def functionname([formal_args,] *var_args_tuple ):
   "函数_文档字符串"
   function_suite
   return [expression]

这里加了 * 说明这个被*修饰的变量会存放所有未命名的参数：

def printinfo( arg1, *vartuple ):
   "打印任何传入的参数"
   print "输出: "
   print arg1
   for var in vartuple:
      print var
   return


# 调用printinfo 函数
printinfo( 10 )
printinfo( 70, 60, 50 )

变量的作用域

python中两个最基本的变量作用域：全局变量和局部变量。

最简单的区分方式：定义在函数内部的变量具有局部作用域， 定义在函数外部的变量具有全局作用域。 全局变量可以在整个程序范围内被访问。局部变量只能在被定义的函数的作用域内访问。

全局变量想作用于函数内部，需要添加 global， 其作用是将一个变量定义为全局变量。

globvar = 0

def set_globvar_to_one():
    global globvar    # 使用 global 声明全局变量
    globvar = 1

def print_globvar():
    print(globvar)     # 没有使用 global

set_globvar_to_one()
print  globvar        # 输出 1
print_globvar()       # 输出 1，函数内的 globvar 已经是全局变量

function domain

函数内部被调用的变量处于Local Domian(局部作用域)，在函数之外被赋值的变量属于Global Domain(全局作用域)

作用域的特点：

  1. 全局作用域代码不能使用局部变量
  2. 局部作用域的代码可以访问全局变量（read-only）
  3. 不同局部作用域之间的代码相互隔离
  4. 不同作用域中的变量名可以相同

综上所述，最好不要用全局变量，尽量使用局部作用域的变量。

Dictionary

introduction

dict and list

字典可以看作为一种List， 字典的Index不仅仅时整数，还可以使用不同的数据类型。字典的Index可以作为“key”， 其相关连的值形成 {Key:Value}对。

myCat = {'size': 'fat', 'color': 'gray'}
>>> myCat['size']
'fat'

字典和列表不同的一点：字典中没有“顺序”的概念，因此无法切片，无法使用数字下标访问对应数字位置的value。

判断两个字典是否相同时，顺序不影响判断结果，但是列表却会受到影响

keys(), values(), items()

返回字典的键， 数值， 键值对，其数据类型分别为：

dict_keys, dict_values, dict_items 与列表不同的是，这些返回值不是真正的列表，因此无法被修改，也没有 append()方法

但是，这些数据类型可以用于 for loop：

spam = {'color' : 'red', 'age' : '42'}

for k in spam.keys():
	print(k)
for v in spam.values():
	print(v)
for i in spam.items():
	print(i)

如果需要返回一个列表，需要使用list()函数

>>>list(spam.keys)
['color', 'age']

也可以使用多重赋值来将key 和 value赋值给不同的变量

for k, v in spam.items():
	print('Key：' + k + 'Value' + str(v))

get()

字典中，访问一个key对应的value， 如果key存在，那么直接访问对用value， 如果key不存在，那么访问get()设置的默认值。【不会改变dict】

get(key, defaultValue)

spam.get('eggs', 0)

可以保证在进行dict访问时不出现错误，同时输出默认value

setdefault()

为某一个key设置默认值， 当这个key没有对应的value时，使用这个默认值，并将其添加到dict中。【会改变dict】


spam = {'name' : pika, 'age' : '42'}
spam.setdefault('color', 'black')
>>>spam
{'color': 'black''name' : pika, 'age' : '42'}

#对已经存在的key设置默认value，不会改变dict
spam.setdefault('color', 'whithe')
>>>spam
{'color': 'black''name' : pika, 'age' : '42'}
